# Ürün Geliştirme Projelerinde Veri Analitiği Kullanılarak Özellik Önceliklendirme ve Risk Değerlendirme Modeli

Bu proje, Amazon Sales Dataset kullanılarak ürün geliştirme süreçlerinde özellik önceliklendirme ve alternatif ürün değerlendirmesi amacıyla hazırlanmıştır.

## Kullanılan Yöntemler

* Analitik Hiyerarşi Süreci (AHP)
* TOPSIS (Technique for Order Preference by Similarity to Ideal Solution)

## Kullanılan Kriterler

* Kullanıcı Memnuniyeti (Rating)
* Kullanıcı İlgisi (Rating Count)
* İndirim Avantajı (%)
* İndirimli Fiyat
* Normal Fiyat

## Kullanılan Alternatifler

* Wayona USB Cable
* Ambrane USB Cable
* Sounce USB Cable
* boAt USB Cable
* Portronics USB Cable

## Sonuç

AHP yöntemi ile kriter ağırlıkları belirlenmiş, ardından TOPSIS yöntemi ile alternatif ürünler sıralanmıştır. Analiz sonucunda boAt USB Cable en yüksek performans skoruna sahip ürün olarak belirlenmiştir.
